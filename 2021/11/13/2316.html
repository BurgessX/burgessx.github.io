常见加密解密方式

分类

  • 取盐校验（不可逆）
    md5 md2 md4 带密码的md5（hmac）
    sha1 sha256 sha512
  • 对称加密
    AES DES 3DES
  • 非对称加密
    RSA（公钥、私钥）同一个明文可以生成不同的密文

判别

  • 根据位数判断
    • md5：16位 32位 40位
    • sha1：40位
    • sha256：64位
    • sha512：128位
  • 根据密文字符判断
    • base64：A-Z a-z 0-9 + _ =
  • 根据已知明文密文对判断
    • 例如：明文输入123456，然后看密文

定位

因为MD5的默认key为0123456789abcdef，所以搜索0123456789abcdef即可定位到加密的位置

伪造浏览器环境

JS 逆向策略：

  • 简单的网站：截取 JS
  • 复杂的网站：伪造环境

浏览器的基本功能：

  • DOM（Document Object Model）渲染引擎

    • HTML 增删改查 API，如 document.createElement(“div”)
    • HTML 数据格式
    • 渲染 Canvas

  • BOM（Brower Object Model）

    • 浏览器实现的一些类，如 Location、Navigator
    • 方法：openssl 提供的 Base64（btoa 等）、AES、DES、MD5

  • JS 引擎

    JS 是一种解释性语言，但官方没有提供解释器，只制定了标准（ECMAScript），而 JS 引擎就是其他公司提供的 JS 解释器

    • 类型：谷歌 V8，微软查克拉， ffg 的 quickjs
    • JSON

由于 JS 引擎中不包含 DOM 和 BOM，因此需要伪造或关联 DOM 和 BOM 对象。

注意关联的 BOM 对象只读。

  1. 要伪造什么
    • 全部伪造：Python 的 jsdom 库（被检测），nodejs（被检测）
    • 部分伪造：某些网站只用到部分对象
      • 如何知道网站用到什么对象
        • 浏览器中调试
        • 本地环境中异常捕获
