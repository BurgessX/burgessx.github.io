HTML 学习笔记

HTML 认知

  • 内核不一样，渲染效果不一样
  • Web 标准的构成
    • 结构：HTML (Hyper Text Markup Language, 超文本标记语言)，负责处理页面元素
    • 表现：CSS，负责处理页面效果
    • 行为：JavaScript，负责处理页面交互
  • Web 标准要求页面实现结构、表现、行为三层分离

第一个 HTML：

HTML 骨架结构：

  • html 标签：网页的整体
  • head 标签：网页的头部
  • body 标签：网页的主体

手写 HTML 骨架：

  • title 标签的内容会在标签中显示
  • HTML 骨架架构不需要手写，在 VS Code 新建 html 文档，输入 ! 后按回车即可自动生成。

开发工具：VS Code、WebStorm、Dreamweaver、Hbuilder

语法

注释

快捷键：Ctrl + /

标签

结构：开始标签 + 包裹的内容 + 结束标签

标签分类：

  • 双标签
  • 单标签：无结束标签，如 <br>（换行）、<hr>（水平线）

标签与标签之间的关系

  • 嵌套关系（父子关系）
  • 并列关系（兄弟关系）

标签学习

在 VS Code 中可直接输入标签名后按回车，即可自动补全，无需手动输入 < 和 >

标题标签

标题标签一共六级

  • 复制粘贴快捷键：Ctrl + C 后不需要换行，直接 Ctrl + V
  • 或者 Shift + Alt + ↓
  • 复制完后如何将数字“加一”：选中 1 后 Ctrl + D 两次，然后按 2

段落标签

Paragraph

代码太长可 查看 -> 自动换行

换行标签

Break

水平线标签

Horizontal Rule

文本格式化标签

  • 加粗：b (bold) 或 strong
  • 下划线：u (underline) 或 ins (insert)
  • 倾斜：i (italic) 或 em (emphasize)
  • 删除线：s (strike) 或 del (delete)
  • 两种方法效果完全一样
  • 语义化更强的内容用后者

图片标签

  • src：图片原路径
  • alt：当图片错误时显示的替换文本
  • title：鼠标悬停显示的标题
  • width：只写 width 不写 height 即为等比例缩放

关于属性

  • 比如 src="cat.gif" 叫属性，src 是属性名，"cat.gif" 是属性值
  • 属性之间用空格隔开
  • 属性没有先后顺序之分

关于路径

  • 绝对路径（了解）：一般从盘符或根路径 / 开始，也可以是完整的网址
  • 相对路径（常用）
  • 实测 / 和 \ Edge 浏览器都能解析

音频标签

  • src：音频路径
  • controls：显示播放的控件
  • autoplay：自动播放（部分浏览器不支持，比如 Edge 和 Chrome）
  • loop：循环播放

音频标签目前支持三种格式：mp3、wav、ogg

视频标签

  • src：视频路径
  • controls：显示播放的控件
  • autoplay：自动播放（Chrome 浏览器可以配合 muted 实现静音自动播放）
  • loop：循环播放
  • 问题：为什么 HTML 中音频和视频是双标签，而图片是单标签？
  • 因为 audio 和 video 标签是 HTML 5 的新标签，当遇到不支持它们的老浏览器，可以通过在开始标签和结束标签之间的文本内容提供提示。

链接标签

  • href：Hypertext Reference，超文本引用
    • 注意 https:// 不能省略，不然会识别为本地链接
  • target：
    • _self：在当前窗口中打开（默认）
    • _blank：在新窗口中打开

HTML 基础

列表标签

列表标签分类：

  • 无序列表
  • 有序列表
  • 自定义列表

无序列表

标签名：

  • ul (Unordered List)
  • li (List)

快捷键：Ctrl + Enter 快速新增一行

注意：

  • ul 标签中只允许包含 li 标签，下同
  • li 标签中可包含任意内容，下同

有序列表

标签名：

  • ol (ordered list)
  • li (list)

自定义列表

自定义列表能实现类似下图的效果

标签名：

  • dl (Definition List)：自定义列表
  • dt (Definition Term)：自定义项
  • dd (Definition Description)：自定义描述

注意：dd 前会默认显示缩进效果

表格标签

标签名：

  • table
  • caption：表格大标题
  • th (Table Head Cell)：表头单元格（默认会加粗文字）
  • tr (Table Row)：行
  • td (Table Detail)：单元格

属性：

  • border：边框宽度
  • width
  • height

注意：实际开发中，width 和 height 属性建议在 CSS 文件中设置

表格结构标签

结构标签

  • thead：表格头部
  • tbody：表格主体
  • tfoot：表格底部

合并单元格

步骤：

  1. 明确合并哪几个单元格

  2. 通过左上原则，确定保留谁删除谁（保留左边或上面的）

  3. 给保留的单元格设置属性，属性值为合并单元格的个数

    • rowspan：跨行合并

    • colspan：跨列合并

注意：不能跨结构（表格头部、主体和底部）合并

表单标签

form、input

input 标签

input 是单标签，可根据 type 属性值展示不同的效果

type 属性值：

  • text：文本框
    • placeholder：占位符
  • password：密码框
    • placeholder：占位符
  • radio：单选框
    • name：联系多个单选框，使它们只能选一个
    • checked：默认勾选
  • checkbox：多选框
    • name：联系多个多选框
    • checked：默认勾选
  • file：文件选择，用于上传文件
    • multiple：可选择多个文件
  • submit：提交按钮，默认显示“提交”二字
  • reset：重置按钮（清空框内的内容），默认显示“重置”二字
  • button：普通按钮，默认不显示文字

示例：

button 标签

功能类似 type="button" 的 input 标签，但是双标签

type 属性值

  • submit：提交按钮（默认），默认不显示文字
  • reset：重置按钮，默认不显示文字
  • button：普通按钮，默认不显示文字

注意：

  • 谷歌浏览器汇总 button 默认是提交按钮

select 标签

用于制作下拉菜单

相关标签：

  • select
  • option

textarea 标签

文本域标签

属性：

  • cols：文本域内可见宽度
  • rows：文本域内可见行数

注意：

  • 右下角可以拖拽改变大小
  • 实际开发时针对于样式效果推荐用 CSS 设置

label 标签

用法：绑定内容与表单标签的关系

用法一：

  • 使用 label 标签把内容（比如文本）包裹起来
  • 在表单标签上添加 id 属性
  • 在 label 标签的 for 属性中设置对应的 id 属性值

用法二：

  • 直接使用 label 标签把内容和表单标签一起包裹起来，无需使用 for 属性

例：绑定单选框和文本。效果：点击文本也能选中对应的单选框

语义化标签

没有语义的布局标签：div 和 span

div 自动换行，span 不会换行

有语义的布局标签（HTML5，主要用于手机端）：

  • header：网页头部
  • nav：网页导航
  • footer：网页底部
  • aside：网页侧边栏
  • section：网格区块
  • article：网页文章
